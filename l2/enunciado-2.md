# Ordenacao turbo
Dada uma sequência de números inteiros, ordene-os em ordem crescente


## Input Specification
Um inteiro t (t ≤ 10⁶) — quantidade de números
t linhas, cada uma contendo um inteiro N (0 ≤ N ≤ 10⁶)

## Output Specification
Imprimir os números em ordem não decrescente, um por linha.

## Specs
Time Limit: 1 second  
Memory Limit: 768 MB  
Output Limit: 4 MB  

---
#### Sample Input #1
    5
    5
    3
    6
    7
    1
#### Sample Output #1
    1
    3
    5
    6
    7
